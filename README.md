# 📊 Python CSV Data Integration — World Happiness Dataset

🚀 Automated integration of multiple yearly datasets into a single structured dataset using Python for efficient data analysis.

---

## 🔍 Overview

This project focuses on **merging and processing multiple CSV/Excel files** containing global happiness data (2015–2024) into a single unified dataset.

The goal was to automate data handling and prepare a clean dataset for analysis, enabling easier exploration of trends in global well-being.

---

## 💼 Key Skills Demonstrated

- Data Integration  
- Data Cleaning & Preprocessing  
- File Handling in Python  
- Automation of repetitive tasks  
- ETL (Extract, Transform, Load)  
- Data Processing using pandas  

---

## ⚙️ Process

- Loaded multiple yearly datasets (2015–2024)  
- Standardized column names and structure  
- Merged datasets into a single file  
- Handled inconsistencies across files  
- Exported final cleaned dataset  

---

## 📂 Dataset Description

The dataset is sourced from the World Happiness Report and includes:

- Ranking  
- Country  
- Regional Indicator  
- Happiness Score (Ladder score)  
- GDP per capita  
- Social support  
- Healthy life expectancy  
- Freedom to make life choices  
- Generosity  
- Perceptions of corruption
